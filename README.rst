========
s2p-view
========

------------------
S-Parameter Viewer
------------------
License: `GPL-3.0-or-later <COPYING>`_

View and plot 1 or 2-port S-parameter (Touchstone) files

.. image:: screenshot.png

Dependencies
============
Python 3, matplotlib (Qt5Agg backend), pyqt5 (gui, widgets), scikit-rf


Building
========
You also need make and pyuic5::

   $ make
